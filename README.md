# Asmi Goyal — Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and interests in **UI/UX design** and **frontend web development**.

The portfolio is designed with a clean, minimal aesthetic and focuses on usability, clarity, and real-world problem solving.

---

##  Features

- Clean and modern UI with a soft green theme  
- Responsive single-page layout  
- Sections for About, Skills, Projects, and Contact  
- Interactive buttons and subtle background elements  
- Easy navigation and readable design  

---

## Tech Stack

- HTML5  
- CSS3  
- Google Fonts (Poppins)

---

##  Sections Included

- **Home / Hero** – Introduction and role  
- **About Me** – Short background and interests  
- **Skills** – UI/UX, Frontend, Programming, and Data Analytics  
- **Projects** – UI/UX case studies and frontend projects  
- **Contact** – Email and LinkedIn details
